# Projetos de Estruturas em C

Este repositório contém três programas em C que utilizam estruturas para realizar tarefas específicas. Abaixo está uma breve descrição de cada programa e instruções básicas para usá-los.

## 1. Estrutura de Aluno



Este programa lê os dados de cinco alunos e determina qual aluno tem a maior média geral das notas das provas. O programa exibe o nome e as notas desse aluno.


## 2. Estrutura de Hora



Este programa lê um conjunto de cinco horas e determina a maior hora entre elas. A maior hora é exibida ao final.



## 3. Estrutura de Pessoa



Este programa lê os dados de seis pessoas e identifica a pessoa mais nova e a mais velha com base na data de nascimento. Os nomes dessas pessoas são então exibidos.
